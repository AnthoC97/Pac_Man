# Fonctionnement du Menu
* Menu principal mène au menu de selection de niveau ou permet de quitter le jeu
* Menu de Selection permet de choisir la map pour la partie
* Il y a 2 map de jeux actuellements répartis sur 2 scènes distinctes

# Fonctionnement de Pacman Battle
* 2 Joueurs s'affrontent sur une map choisi précédemment.
* Celui qui touche la Gumball devient rouge et peut tuer son adversaire en le touchant
* Au bout de 10 secondes la Gumball réaparait et le tueur redevient un simple Pacman
* P pour la Pause

# Choix joueurs
* Human Azerty : ZQSD pour se déplacer
* Human Fleche : Fleche pour se déplacer
